## Tweets & Eleições - NLP 🗳️

Estudo pessoal focado na detecção de padrões de polarização e identificação de discursos hostis no Twitter (X) durante períodos eleitorais, utilizando técnicas de Processamento de Linguagem Natural (NLP).

A coleta dos dados é realizada através da **X Developer API**.

_⚠️ Nota Ética: Todos os dados são coletados e processados respeitando estritamente os Termos de Serviço da plataforma e as diretrizes de privacidade de dados._
